# Audited TodoList
Spring Boot todo list with Auditing

Usage (with eclipse):
1.) Clone the project
2.) Eclipse: File -> Import -> Maven -> Existing Maven Projects
3.) Run
4.) Navigate to localhost:8080

Application has two demo users: 
user/user (role=USER) and
admin/admin (role=ADMIN)
